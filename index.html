How to Contribute 🛠️

Follow these simple steps to add your card and make your first open-source contribution!

01

Fork the Repository

Click the Fork button on the top-right of the GitHub repository page. This creates your own copy of the project.

# After forking, you'll have:
github.com/your-username/team-contribution-project
02

Clone Your Fork

Download your forked copy to your local machine using the git clone command.

git clone https://github.com/your-username/team-contribution-project.git
cd team-contribution-project
03

Create a New Branch

Always create a new branch for your changes. Never push directly to main!

# Use the format: add/your-name
git checkout -b add/your-name
04

Add Your Contributor Card

Open index.html, find the comment that says "ADD YOUR CARD HERE", and paste your card code there. See the template inside the file!

<!-- ===== ADD YOUR CARD HERE ===== -->
<!-- Copy the template above and fill in YOUR details -->
05

Commit & Push

Save your changes, commit them with a proper message, and push to your fork.

git add .
git commit -m "feat: add Your Name contributor card"
git push origin add/your-name
06

Open a Pull Request

Go to the original repository on GitHub and click "Compare & Pull Request". Write a short description and submit!

# PR Title format:
feat: Add Your Name — [Your Role]

📋 Contribution Rules

  • ✅ One card per contributor — keep it about yourself!
  • ✅ Use branch naming: add/your-name
  • 🚫 Never push directly to the main branch
  • 🚫 Do not modify other contributor's cards
  • ✅ Keep your bio under 150 characters
  • ✅ Make sure the page still looks good after your changes
